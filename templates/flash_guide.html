⚠ MEDIATek DEVICE DETECTED
This device uses a MediaTek chipset. Odin on Windows is REQUIRED. Heimdall cannot flash MediaTek devices on macOS/Linux. Use VirtualBox with Windows VM or Boot Camp.
⚠ DEVICE NOT IN DOWNLOAD MODE
Your device must be in Download Mode for Odin flashing. Reboot to Download Mode and refresh.

STEP 1: PREREQUISITES

Complete all prerequisites before proceeding.

STEP 2: ENTER DOWNLOAD MODE

  1. Power off the device completely
  2. Press and hold Volume Down + Power simultaneously
  3. When the warning screen appears, release both buttons
  4. Press Volume Up to confirm Download Mode
  5. Screen should show "Downloading... Do not turn off target!!"
Tip: If device boots normally, repeat the process. Timing is key — hold both buttons before connecting USB.

STEP 3: SELECT CSC TYPE

OptionDataUse When
HOME_CSCPreservedUpdating firmware, keeping apps/data
CSCAll ErasedClean install, region change, selling device
⚠ WARNING: Selecting CSC (not HOME_CSC) will perform a COMPLETE FACTORY RESET. All user data, apps, settings, and accounts will be ERASED.

POST-FLASH

  1. Wait for "PASS!" in Odin (green box)
  2. Device will reboot automatically
  3. First boot may take 3–5 minutes — this is normal
  4. Complete initial setup wizard
  5. Verify firmware: Settings → About phone → Software info
  6. Check CSC: Dial *#1234#
